# 添加设备  
通过添加设备页面，你可以添加设备到你的账号。  
在**我的设备**页面点击右上角的“+”图标，即可进入**添加设备**页面，页面如下：  
![](assets/002/01-1524463170000.png)


对于DIY用户，目前支持的硬件设备有：  
### Arduino  
使用Arduino SDK的开发板：官方Arduino、esp8266开发板、esp32开发板等  

### Linux开发板  
使用Linux操作系统的设备：树莓派、香蕉派等
